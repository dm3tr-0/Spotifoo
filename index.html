🎯 что это?

Spotifoo — альтернативный клиент Spotify для Android, который даёт возможности, недоступные в официальном приложении.

Вдохновлён Spicetify для ПК.

✨ возможности

  • Свободный выбор любого трека
  • Перемотка и пропуск треков без ограничений
  • Блокировка аудио- и баннерной рекламы
  • Свои темы оформления (в планах)

⚙️ как работает

В отличие от "взломанных" APK, Spotifoo не модифицирует оригинальное приложение.

  1. Открывает open.spotify.com в GeckoView
  2. Маскируется под десктопный Firefox
  3. Использует переработанный код расширения Spicetify (Adblockify)
  4. Подключает uBlock Origin для блокировки рекламы
  5. Инжектит CSS для мобильной адаптации интерфейса
